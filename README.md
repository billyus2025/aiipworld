# IP Mother Factory V3.0

## Architecture
This project is structured as a modular factory for generating IP content (Drama, Novel, Comic).

### Directory Structure
- `factory/`: Core logic modules.
- `data/`: Data storage.
- `frontend/`: Frontend templates and assets.
- `backend/`: API routes and utilities.
- `sites/`: Generated site output.
- `config/`: Global configuration.

## Usage
1. Configure `config/factory.json`.
2. Run `frontend/build.js` to build sites.
3. Use `backend/` for API access.
